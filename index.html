Weasyl Projects

Weasyl Projects is a place for code that wouldn't necessarily belong on the Weasyl mainsite. Some projects are third-party code developed specifically for Weasyl, are not official Weasyl products, and do not represent Weasyl views and opinions.

Site look-and-feel

  • Weasyl Enhancer, for Firefox or Chrome, which allows additional image and text information to be displayed when hovering over a thumbnail image. By Noxid.
  • Weasyl Notifier, for Chrome or Safari, which creates an icon showing the number of Weasyl messages you have. By Sunookitsune.

Gallery maintenance

  • Waxpost, a Chrome extension that simplifies crossposting visual submissions from other art sites to Weasyl, by weykent.

User accounts

  • Conbadger, a tool for generating pretty QR code badges that link to Weasyl profiles, by simonfox and weykent.
  • Simple Ottermatic Follower Transfer, a migration assistant for Weasyl users coming from Fur Affinity, by Dos.
