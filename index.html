Weasyl Projects

Weasyl Projects is a place for code that wouldn't necessarily belong on the Weasyl mainsite. Some projects are third-party code developed specifically for Weasyl, are not official Weasyl products, and do not represent Weasyl views and opinions.

Site look-and-feel

  • Weasyl Enhancer, which allows additional image and text information to be displayed when hovering over a thumbnail image. Firefox and Chrome extensions available. By Noxid.
  • Weasyl Notifier, for Chrome or Safari, which creates an icon showing the number of Weasyl messages you have. By Sunookitsune.

Gallery maintenance

  • Art Gallery Manager: Enables artists to manage their galleries offline and batch-upload submissions to multiple websites. Also enables consumers of art to manage galleries of downloaded art. By Kitashi.
  • Waxpost, a Chrome extension that simplifies crossposting visual submissions from other art sites to Weasyl, by weykent.

User accounts

  • Conbadger, a tool for generating pretty QR code badges that link to Weasyl profiles, by simonfox and weykent.
  • Exodus Helper, a migration assistant allowing cross referencing user accounts from Fur Affinity to other sites (including Weasyl), by Kobi Tate. This tool essentially supersedes an older, now unmaintained tool which did much the same thing, the Simple Ottermatic Follower Transfer.
